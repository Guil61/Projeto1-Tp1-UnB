\doxysubsection{Detailed Description}
Classe que representa uma ordem de investimento. 

A classe \doxylink{class_ordem}{Ordem} contém informações sobre uma operação de compra ou venda, incluindo código de identificação, código de negociação, data, preço, quantidade e a carteira à qual está associada.
